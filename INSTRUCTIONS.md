# 配列・繰り返し文

## 1. 次のプログラムを書き換えて、配列の最後から順番に処理するように変更してください。

```c++

	int array[] = {4,2,1,9,5};

	int i = 0;
	while(i < 5)
	{
		cout << array[i] << endl;
		i +=1;
	}
```

## 2. [1.]で書き換えたプログラムをfor文に書き換えてください。
## 3. [1.]で書き換えたプログラムをdo-while文に書き換えてください。
